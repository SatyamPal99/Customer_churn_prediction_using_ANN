# Customer_churn_prediction_using_ANN

## Dataset

Name: Churn_Modelling.csv  
Source: Kaggle  

### Features:
- Credit Score  
- Geography  
- Gender  
- Age  
- Tenure  
- Balance  
- Number of Products  
- Has Credit Card  
- Is Active Member  
- Estimated Salary  

**Target Variable:**  
 Exited` → 1 (Customer left)  
 Exited` → 0 (Customer stayed)
